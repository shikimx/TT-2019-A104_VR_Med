Launch the Share to Facebook modal via a deeplink to Home on Gear VR, allowing users to share local media files to Facebook.

Accepts a postTextSuggestion string for the default text of the Facebook post. Requires a filePath string as the path to the image to be shared to Facebook. This image should be located in your app's internal storage directory. Requires a contentType indicating the type of media to be shared (only 'photo' is currently supported.)

Parameters
postTextSuggestionthis text will prepopulate the facebook status text-input box within the share modal
filePathpath to the file to be shared to facebook
contentTypecontent type of the media to be shared
