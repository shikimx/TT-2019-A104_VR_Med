This method will ensure only one specific type TActivate in a given group of components derived from the same TCategory type is enabled.

This is used by the sample support code to select between different targeting, input, aim, and other handlers.

This generic method is used for activating a specific set of components in the LocomotionController.

This is just one way to achieve the goal of enabling one component of each category (input, aim, target, orientation and transition) that the teleport system requires.
